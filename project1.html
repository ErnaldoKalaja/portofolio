DietDash (February - May 2023) (Dart, Flutter, Firebase, REST API)

  • Created “DietDash”, a diet app that allows users to search for meal recommendations based on calorie count and diet choice.
  • Developed an intuitive and user-friendly interface for the app to provide a smooth user experience.
  • Utilized Flutter's cross-platform capabilities to ensure compatibility with both iOS and Android devices.
  • Implemented features such as calorie tracking, meal planning, and exercise tracking to help users achieve their dietary goals.
  • Designed and integrated a backend system for storing and retrieving user data securely.
